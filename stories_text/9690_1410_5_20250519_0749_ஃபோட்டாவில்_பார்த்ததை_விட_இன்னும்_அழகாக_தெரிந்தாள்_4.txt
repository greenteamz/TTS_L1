அவள் : ஹாய் சமர்..
நான் : எனக்கு சில ப்ராஜெட் ஒர்க் இருந்தது. அதனால வெளியே போறேன்.. இப்ப என்னால் வர முடியாது..
அவள் : ஹே சமர் இத சொல்ல தானா கால் பண்ண… சரி உன் ஓர்க் பாரு..
நான் : இல்ல.. உங்க கூட இருப்பேன் சொன்னேன்ல. அதான்.. நீங்க எதுவும் பீல் பண்ண கூடாது கால் பண்ணேன்.
அவள் : ஓ சரி சரி…
அவள் தன் காலை கட் பண்ணியவுடன் வாட்ஸ் ஆப் எப்படியும் பார்ப்பாள் என்று எனக்கு நன்றாக தெரியும்.. நான் நினைத்தது மாதிரியே அவள் ஆன்லைனில் வந்தாள்.
அவள் : பரவாயில்லை டா..
நான் : ஐயம் வெரி சாரி.. ரியல் சாரி..
அவள் : என்னைய ரொம்ப தப்பானவ நினைக்காத ராகுல்.. நேத்து நீ பேசின வார்த்தையை என்னால் தைரியமா கேட்க கூட முடியல.. நான் ஒன்னும் அந்த அளவுக்கு உடல் சுகத்துக்கு அலையுற ஆள் இல்ல.. உனக்கு தெரியுமா? ஒரு பொண்ணு இது மாதிரியான கஷ்டத்தை கடந்து வரது எவ்வளவு கஷ்டம்னு.. இனி அந்த மாதிரி பேசாத என்னை…
நான் : ஹே.. என்னை நம்பு. அது மாதிரி திரும்பி நடக்கவே நடக்காது. நவ்.. ஐயம் நியூ ராகுல்.. உன்ன நல்ல கவனிச்சுக்கிறேன்.
அவள் : ஹ்ம்ம்.. அப்படியா..
நான் : அவளிடம் பழைய ராகுல் மாதிரியே பேச ஆரம்பித்துவிட்டேன்..
அவள் : ஹி.. ஹி..
நான் : திஸிஸ் மை டியர்…
பாபி : ஆபிஸ்லையா இருக்க?
நான் : நோ… டு டே ஆபிஸ் நான் போகல..
அவள் : டெல்லியில கிளைமேட் எப்படி இருக்கு? இன்னும் குளிரா தான் இருக்கா?
நான்: யா… நடுங்குற அளவுக்கு குளிரா இருக்கு…
நான் ஒரு ராகுல் போல நடித்து அவளிடம் பேசிக் கொண்டிருந்தேன் .. ஹே .. அவன் ஊரு டெல்லி தான் எனக்குத் தெரியாது.
நான் அவளிடம் சிக்க மாட்டேன் என்ற நம்பிக்கையில் பேசுகிறேன்… இதுக்கு முன் பேசினத பத்தி மற்றும் கடந்த காலத்தை ஏதாவது கேட்டால், நான் அவ்வளவு தான்.. தொலைஞ்சேன்..
அவள் : சரி உன்ன நீ சூடாக்கி வைச்சுக்கோ…
நான் : அதுக்கு உன் உதவி கட்டாயம் தேவை… நீ பண்ணுவேன் நம்புறேன்..
அவள் : என் உதவியா? என்ன?
நான் : என்ன வந்து இறுக்கமா கட்டிப்பிடி.. உன் உடம்போடு உரசு சூடு ஆகிடும்..
அவள் : ஐயாவுக்கு என்ன ஆச்சு இன்னிக்கு? சூடாக இருக்கிங்க? அதுவும் காலைல?
அந்த லூசு காலையில் ஒருபோதும் இவளிடம் சேட் செய்யமாட்டான் என்று நினைக்கிறேன்…
நான் : ம்ம்.. ஆமா..
அவள் : அப்படினா உன் ஒயிப் வீட்டில் இருக்காளா? இல்ல.. ஸ்கூல்க்கு வேலைக்கு போய்ட்டாளா?..
அவனுக்கு பொண்டாட்டி வேற இருக்காளா!? அதுவும் எனக்குத் தெரியாது..
நான் : அவள் இல்ல.. வேலைக்கு போய்ட்டா..
அவள் : அப்பிடியா, சோ.. என்னால எப்படி உனக்கு உதவ முடியும்.? ஹிஹி.. .
நான் : இப்ப உன் புண்டைல சுண்ணி விட்டு ஓக்கனும்..
அவள் : ஓ மை காட்.! என்ன இப்படி பச்சையா பேசுற! வயக்ரா க்யாவை எதுவும் எடுத்திடியா?..
ஓ தேவையில்லாமல் பேசிட்டோமா நினைத்தேன்..
நான் : ஐயம் நியூ ராகுல் கியர்.. ரெடியா இரு..
அவள் : ஹே. ஐ லவ் யூ டியர்..
நான் : சோ.. நீ என்ன பண்ணிட்டு இருக்க டியர்?
அவள் : காலைல டிபன் சாப்பிட்டு, டிரஸ் வாஸ் பண்ணலாம் ப்ளான் பண்ணி இருக்கேன்..
நான் : நீ பிறகு டிரஸ் வாஸ் பண்ணிக்கோ… இப்ப கொஞ்ச நேரம் என் கூட டைம் ஸ்பெண்ட் பண்ணு..
அவள் : மை ப்ளஷர் டியர்.
நான் : சோ.. நீ சரியா இப்ப என்ன பண்ணிட்டு இருக்க?
அவள் : பெட்ல படுத்திருக்கேன் டா.. நீ.?
நான் என் ஜன்னலிலிருந்து அவள் பெட்ரூமை எட்டி பார்த்தேன். அவள் மகிழ்ச்சியான முகத்துடன் படுத்து இருந்தாள்..
நான் : சேம் டு மி டியர்..
அவள் : நீ இந்த மாதிரி செக்ஸ் மூடுல காலைல இருக்கமாட்டியே டியர்?
நான் : அரேய் இப்ப அத விடு… ஓகே.. ஈவினிங் மெசேஜ் பண்றேன்..
அவள் : நோ. நோ. நோ.. சாரி.. சாரி…
நான் : ஹ்ம்ம்.
பாபி : சொல்லு டியர்
நான்: ஒன்னுமில்ல..
அவள் : ஹ்ம்ம் கோவமா?
நான் : ஹ்ம்ம்..
அவள் : கோவலத்துல கூட அழகாக தான் இருக்க..
நான் : ஹ்ம்ம்..
அவள் : ஐய்யோ.. உன் கோவம் போக என்ன பண்ணனும்.?
நான் : உன் ஃபோட்டா அனுப்பு..
அவள் : இது தானா.. வெயிட் பண்ணு..
அவள் பெட்டில் படுத்துக் கொண்டே ஒரு செல்ஃபி எடுத்து அனுப்பினாள். அவள் மிகவும் கவர்ச்சியாக இருந்தாள்.
நான் : வாவ் .. சூப்பராக இருக்க..
அவள் : தாங்கஸ்.
நான் : நீ என்ன டிரஸ் போட்டுட்டு இருக்க?
அவள் : ஃபோட்டால நீ பாத்தில? அதான்.. நைட்டி, பிங்க் ப்ரா, பேன்டி
நான் : ஃபோட்டால ப்ரா பேன்டி தெரியல..
அவள் : ஹே… யூ நாட்டி..
நான் : உன் பக்கத்துல வரலாமா?
அவள் : பிளிஸ் டியர். கிட்ட வா..
நான் : யா வந்து. உன் கையை பிடிக்கறேன். இட்ஸ் சோ சாப்ட்.
அவள் : ஓ சூப்பர்.. அருமை
நான் : உன் கைல கிஸ் பண்றேன். உம்மாஹ் உம்மாஆஆஆ
அவள் : ஆ. டார்லிங்..
நான் : இப்ப உன் கண்ண பாக்குறேன். அது ரொம்ப ஏரோடிக்கா இருக்கு..
பாபி : நீ இப்படி பேசுறது ரொம்ப சந்தோஷமா இருக்குடா..
நான் : உன் லிப்ஸ் பக்கத்துல வரேன்
அவள் : உம்ம்மாஹாஹ் உம்மாஹாஹ..
நான் : உம்மாம்மாம்மாம்மா நீண்ட முத்தம்..
அவள் : ஐ லவ் யூ சோ மச் டியர்..
நான் : மீ டு டார்லிங்.. பிளீஸ் உன் நாக்க நீட்டு..
அவள் : யா.. டேக் இட்..
நான் : மெதுவா அதை சப்பி உறுஞ்சுறேன். ஆஹா. உன் நாக்கு டேஸ்ட்டா இருக்கு..
பாபி : ஹ்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்ம்.
நான் : உன் உமிழ்நீரை டேஸ்ட் பண்றேன்
அவள் : உன் நாக்கும் டேஸ்ட்டா இருக்கு.
நான் : மெதுவா லைட் கடிக்குறேன்.
அவள் : ஆஹ்ஹ்ஹ்ஹ் நீ என்னை கொல்றியேடா..
நான் : இப்ப உன் கழுத்தில கிஸ் பண்றேன்.. உம்ம்மாமா ..
அவள் : அஹ்ஹ்ஹ்ஹ்
நான் : பேபி
பாபி : யா டியர்
நான் : இப்ப நைட்டியை ரிமூவ் பண்றேன்..
அவள் : ஏய்.. இது டே டைம்.. டியர். எனக்கு வெட்கமா இருக்கு..
நான் : சரி
அவள் : சொல்லு
நான் : நத்திங்.
அவள் : ஹே .. கோவமா? வெயிட் பண்ணு.. ரிமூவ் பண்றேன்..
நீங்களும் காத்திருங்கள்..
இனியும் வருவாள்..
